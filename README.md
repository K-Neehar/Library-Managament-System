# Library-Management-System

This is an application that will let students add and display books.  This application will be user-friendly and effortless to access and use. It also lets users check and issue their favorite books anytime, anywhere with one click. 

This is an desktop application for the library management system using a C++ programming language. 

Features required in the Library Management system:

1. Login system
2. Enter, delete, and update student/
3. Check details of existing data
4. Check for availability of books subject wise
5. View all the books and quantity
